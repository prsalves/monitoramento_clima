# 🌦 Monitoramento de Clima com API

## 📌 Sobre o projeto

Este projeto consiste em uma aplicação desenvolvida em Python que realiza a consulta de dados meteorológicos por meio de uma API de clima. O sistema envia requisições para um serviço externo, recebe as informações em formato JSON e apresenta dados relevantes sobre as condições climáticas de uma cidade.

## ⚙️ Tecnologias utilizadas

* Python
* Integração com API externa
* Requisições HTTP
* Manipulação de dados JSON

## 🎯 Objetivo

O objetivo deste projeto é praticar conceitos importantes de desenvolvimento em Python, como integração com APIs externas, consumo de serviços web e processamento de dados retornados por requisições HTTP.
